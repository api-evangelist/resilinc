# Resilinc (resilinc)
Resilinc as a Cloud / SaaS based platform provider offers supply chain API software and provides easy access to vast data via REST APIs. These APIs help your developers access the raw dataset, which can either be used for offline analysis or supply chain API integration with their IT systems.
